# Demo Video Narration Script

> ElevenLabs voiceover script for the 2-minute demo video.
> Settings: Speed 0.9, calm male voice (e.g. Adam or Antoni)

---

Have you ever written a social media post and wondered —
"Who is this actually going to resonate with?"

Crafting content that speaks to casual readers,
aspiring entrepreneurs, and technical audiences
all at once — takes a lot of time and effort.

This system solves that automatically.

First, select your target platform.
You can choose from X, formerly Twitter, note dot com, or Amazon KDP.

For this demo, we'll go with X.

You can also switch the output language between English and Japanese.
The passing score threshold is set at 70 out of 100.

Now, let's enter our content theme.

"How I Make $10,000 per month on the Side Using AI"

Hit Generate, and the Generator Agent immediately
drafts an initial post optimized for X.

Three Persona Agents then evaluate it in parallel.

The General persona represents casual readers.
The Side Biz persona represents aspiring entrepreneurs.
The Tech persona represents technical audiences.

Each agent returns a score from 0 to 100,
along with specific improvement feedback.

If any persona scores below the threshold,
the Rewriter Agent automatically rewrites the content
and the evaluation loop runs again.

You can open the PDCA Journey to see exactly
what happened in each loop.

Once all three personas pass,
the final content is ready.

Here's the result — a polished X post
that resonates across all three audience types.

With the Multi-Persona Content PDCA Agent,
quality-checking a single post takes just seconds —
not minutes.

Powered by Microsoft Azure AI Foundry
and the Microsoft Agent Framework.
